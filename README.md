🍊 Swiggy Sales Analytics Dashboard (Excel)
An interactive business intelligence dashboard built entirely in Microsoft Excel, analysing ₹53M+ in Swiggy food delivery sales across India.

Dashboard Preview

📌 Project Overview
This project demonstrates how Microsoft Excel can be used as a full-fledged BI tool — without any external software. It covers end-to-end data analysis for a food delivery platform with 197K+ orders across multiple Indian cities and states.

📊 Key Metrics Tracked
Metric	Value
Total Sales	₹53.01M
Total Orders	197K
Avg Order Value	₹268.51
Average Rating	4.34
Rating Count	5.59M
🔍 Dashboard Features
KPI Summary Cards — Quick snapshot of overall business health
Monthly Sales Trend — Line chart showing seasonal sales patterns
Daily Sales Trend — Bar chart for day-of-week performance analysis
Weekly Sales Trend — 36-week performance overview
Top 5 Cities by Sales — Horizontal bar chart comparison
Sales by Food Type — Veg vs Non-Veg donut chart
State-wise Sales Map — Geographic heatmap using Excel Maps (Bing)
Quarter-wise Summary Table — Q1/Q2/Q3 with sales, ratings, and orders
Interactive Slicers — Filter by month, food category, and restaurant

Q2 is the peak sales quarter at ₹19.90M
Average order value stands at ₹268.51 across 197K+ orders
All three quarters maintain a consistent rating of 4.3
